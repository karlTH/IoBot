IoBot 3D printed parts.


Description:

3D printed elements to build a robot IoBot.


Installation process:

Install Blender:
http://www.blender.org/


Instructions:

This file contains all the elements which are both included in the robot. 
For better clarity some items were partially hidden in the Outliner.
